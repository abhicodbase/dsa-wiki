# Count Number of Nice Subarrays

- **Difficulty:** Medium
- **Categories:** Array, Hash Table, Math, Sliding Window, Prefix Sum
- **Time Complexity:** O(N)
- **Space Complexity:** O(N)

---

Count subarrays with exactly k odd numbers.

---

## Approach: Prefix Count (At Most K - At Most K-1)

Count subarrays with AT MOST k odds and subtract AT MOST k-1 odds. 'At most k odds' computed with sliding window tracking odd count.
