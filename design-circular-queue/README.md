# Design Circular Queue

- **Difficulty:** Medium
- **Categories:** Array, Linked List, Design, Queue
- **Time Complexity:** O(1)
- **Space Complexity:** O(K)

---

Design a Circular Queue with enQueue, deQueue, Front, Rear, isEmpty, isFull operations.

---

## Approach: Array with Head/Tail Pointers

Use a fixed-size array of capacity k+1 (or k with a size counter). Head and tail pointers move circularly. Size tracks number of elements.
