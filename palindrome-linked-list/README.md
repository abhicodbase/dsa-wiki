# Palindrome Linked List

- **Difficulty:** Easy
- **Categories:** Linked List, Two Pointers, Stack, Recursion
- **Time Complexity:** O(N)
- **Space Complexity:** O(1)

---

Given the head of a linked list, return true if it is a palindrome.

---

## Approach: Find Middle + Reverse Second Half

Use slow/fast pointers to find the middle. Reverse the second half in-place. Compare first and second halves node by node. Restore (optional) and return result.
