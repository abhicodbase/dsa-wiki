# Sort an Array

- **Difficulty:** Medium
- **Categories:** Array, Divide and Conquer, Sorting, Heap, Merge Sort
- **Time Complexity:** O(N log N)
- **Space Complexity:** O(N)

---

Sort an array of integers in ascending order. Implement Merge Sort or Quick Sort without built-in sort functions.

---

## Approach: Merge Sort

Divide array into halves recursively. Merge sorted halves using two pointers into a scratch array. Guaranteed O(N log N) even in worst case.
