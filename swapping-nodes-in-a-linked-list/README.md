# Swapping Nodes in a Linked List

- **Difficulty:** Medium
- **Categories:** Linked List, Two Pointers
- **Time Complexity:** O(N)
- **Space Complexity:** O(1)

---

Swap the values of the k-th node from the beginning and the k-th from the end.

---

## Approach: Two Pointers

Find k-th from front. Use fast/slow technique to find k-th from end. Swap their values (not nodes) — O(1) value swap.
