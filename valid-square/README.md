# Valid Square

- **Difficulty:** Medium
- **Categories:** Math, Geometry
- **Time Complexity:** O(1)
- **Space Complexity:** O(1)

---

Determine if four points form a valid square (equal sides, equal diagonals, all non-zero).

---

## Approach: Distance Multiset Check

Compute all 6 pairwise distances. A valid square has exactly 2 distinct distances: 4 equal sides and 2 equal diagonals (longer). Check this condition.
