# Invert Binary Tree

- **Difficulty:** Easy
- **Categories:** Tree, Depth-First Search, Breadth-First Search, Binary Tree
- **Time Complexity:** O(N)
- **Space Complexity:** O(H)

---

Invert a binary tree (mirror it). Return the root.

---

## Approach: Recursive DFS

Swap left and right children at each node. Recursively invert both subtrees. Base case: null node returns null.
