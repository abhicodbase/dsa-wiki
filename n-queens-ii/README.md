# N-Queens II

- **Difficulty:** Hard
- **Categories:** Backtracking
- **Time Complexity:** O(N!)
- **Space Complexity:** O(N)

---

Return the number of distinct solutions to the N-Queens puzzle.

---

## Approach: Backtracking (Count Solutions)

Same as N-Queens backtracking but just increment a counter at each valid configuration instead of recording the board.
