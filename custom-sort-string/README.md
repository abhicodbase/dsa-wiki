# Custom Sort String

- **Difficulty:** Medium
- **Categories:** Hash Table, String, Sorting
- **Time Complexity:** O(N + M log M)
- **Space Complexity:** O(N)

---

Sort string s using the custom order defined by string order where each character appears once.

---

## Approach: Count + Reconstruct

Count character frequencies in s. Output characters in the order defined by order string (with their counts). Append remaining characters not in order in any order.
