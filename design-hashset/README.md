# Design HashSet

- **Difficulty:** Easy
- **Categories:** Array, Hash Table, Linked List, Design, Hash Function
- **Time Complexity:** O(1) avg
- **Space Complexity:** O(N)

---

Design a HashSet from scratch with add(key), contains(key), remove(key). No built-in hash set libraries.

---

## Approach: Array of Buckets

Same as HashMap but store only keys (no values). Use chaining within each bucket to resolve collisions.
