# 4Sum

- **Difficulty:** Medium
- **Categories:** Array, Two Pointers, Sorting
- **Time Complexity:** O(N³)
- **Space Complexity:** O(N)

---

Given an array `nums` and a target, find all unique quadruplets that sum to target.

---

## Approach: Sort + Three Pointers

Sort the array, fix two outer pointers with deduplication, then use a classic two-pointer approach for the inner pair. Skip duplicates at each level.
