# Largest Number

- **Difficulty:** Medium
- **Categories:** Array, String, Sorting, Greedy
- **Time Complexity:** O(N log N)
- **Space Complexity:** O(N)

---

Given a list of non-negative integers, arrange them to form the largest number.

---

## Approach: Custom String Comparator

Sort using comparator: for two numbers a and b as strings, compare ab vs ba. Sort descending. Handle edge case of all zeros.
