# Valid Anagram

- **Difficulty:** Easy
- **Categories:** Hash Table, String, Sorting
- **Time Complexity:** O(N)
- **Space Complexity:** O(1)

---

Given two strings s and t, return true if t is an anagram of s.

---

## Approach: Character Frequency Array

Count character frequencies for s (increment) and t (decrement) in a 26-element array. If all counts are zero, they're anagrams.
