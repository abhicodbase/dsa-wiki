# Sqrt(x)

- **Difficulty:** Easy
- **Categories:** Math, Binary Search
- **Time Complexity:** O(log N)
- **Space Complexity:** O(1)

---

Given a non-negative integer x, compute and return the square root, rounded down to the nearest integer.

---

## Approach: Binary Search

Binary search in [0, x]. Find largest mid where mid*mid <= x.
