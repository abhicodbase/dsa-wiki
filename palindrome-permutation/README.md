# Palindrome Permutation

- **Difficulty:** Easy
- **Categories:** Hash Table, String, Bit Manipulation
- **Time Complexity:** O(N)
- **Space Complexity:** O(1)

---

Given a string, return true if a permutation of it can form a palindrome.

---

## Approach: Odd Frequency Count

A string can form a palindrome if at most one character has an odd frequency. Use a set: toggle characters in/out. If size ≤ 1 at end, return true.
