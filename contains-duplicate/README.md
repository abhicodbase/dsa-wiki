# Contains Duplicate

- **Difficulty:** Easy
- **Categories:** Array, Hash Table, Sorting
- **Time Complexity:** O(N)
- **Space Complexity:** O(N)

---

Given an integer array, return true if any value appears at least twice, false if every value is distinct.

---

## Approach: Hash Set

Add each number to a hash set. If the number already exists in the set, return true (duplicate found). Return false after full traversal.
