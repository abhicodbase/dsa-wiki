# Same Tree

- **Difficulty:** Easy
- **Categories:** Tree, Depth-First Search, Breadth-First Search, Binary Tree
- **Time Complexity:** O(N)
- **Space Complexity:** O(H)

---

Given two binary trees, check if they are the same (structurally identical with same node values).

---

## Approach: Recursive DFS

Both null → true. One null → false. Values differ → false. Recurse on left and right subtrees. Both must match.
