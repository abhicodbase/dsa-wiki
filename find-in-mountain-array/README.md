# Find in Mountain Array

- **Difficulty:** Hard
- **Categories:** Array, Binary Search, Interactive
- **Time Complexity:** O(log N)
- **Space Complexity:** O(1)

---

Find target in a mountain array using the MountainArray interface's get(index) method. Binary search is necessary — direct access is limited.

---

## Approach: Binary Search in 3 Phases

Phase 1: Find peak index. Phase 2: Binary search ascending part. Phase 3: Binary search descending part if not found.
