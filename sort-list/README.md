# Sort List

- **Difficulty:** Medium
- **Categories:** Linked List, Two Pointers, Divide and Conquer, Sorting, Merge Sort
- **Time Complexity:** O(N log N)
- **Space Complexity:** O(log N)

---

Sort a linked list in O(N log N) time and in-place.

---

## Approach: Bottom-Up Merge Sort

Divide list into halves using slow/fast pointers, recursively sort each half, then merge. Bottom-up approach avoids O(N) stack space.
