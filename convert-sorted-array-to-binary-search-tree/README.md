# Convert Sorted Array to BST

- **Difficulty:** Easy
- **Categories:** Array, Divide and Conquer, Tree, Binary Search Tree, Binary Tree
- **Time Complexity:** O(N)
- **Space Complexity:** O(log N)

---

Given a sorted array, convert it to a height-balanced BST.

---

## Approach: Divide and Conquer (Midpoint as Root)

Recursively pick the midpoint of the current range as the root. Left half becomes left subtree, right half becomes right subtree.
