# Merge Two Sorted Lists

- **Difficulty:** Easy
- **Categories:** Linked List, Recursion
- **Time Complexity:** O(M+N)
- **Space Complexity:** O(1)

---

Merge two sorted linked lists and return it as a new sorted list.

---

## Approach: Iterative with Dummy Head

Use a dummy head node. Compare heads of both lists; attach the smaller to the result list. Advance the pointer of the chosen list. Attach remaining nodes at the end.
