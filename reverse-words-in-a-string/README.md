# Reverse Words in a String

- **Difficulty:** Medium
- **Categories:** Two Pointers, String
- **Time Complexity:** O(N)
- **Space Complexity:** O(N)

---

Given a string s, reverse the order of words (split by spaces), trimming leading/trailing spaces and collapsing multiple spaces.

---

## Approach: Split and Reverse

Split string by whitespace (handles multiple spaces automatically), filter empty tokens, reverse the list, join with single space.
