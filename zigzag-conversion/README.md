# Zigzag Conversion

- **Difficulty:** Medium
- **Categories:** String
- **Time Complexity:** O(N)
- **Space Complexity:** O(N)

---

Write the string in a zigzag pattern on a given number of rows, then read row by row.

---

## Approach: Row Simulation

Simulate the zigzag by distributing characters into numRows string buckets, tracking current row and direction. Concatenate all rows at the end.
