# Single Number

- **Difficulty:** Easy
- **Categories:** Array, Bit Manipulation
- **Time Complexity:** O(N)
- **Space Complexity:** O(1)

---

Every element appears twice except one. Find that one element without extra memory.

---

## Approach: XOR

XOR all elements together. Duplicate pairs cancel out (a XOR a = 0). The result is the unique element.
