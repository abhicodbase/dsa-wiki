# Ugly Number

- **Difficulty:** Easy
- **Categories:** Math
- **Time Complexity:** O(log N)
- **Space Complexity:** O(1)

---

An ugly number is a positive number whose prime factors are limited to 2, 3, and 5. Determine if n is ugly.

---

## Approach: Divide by 2, 3, 5 Repeatedly

Divide n by 2, 3, and 5 as long as divisible. If result is 1, n is ugly. If n ≤ 0, return false.
